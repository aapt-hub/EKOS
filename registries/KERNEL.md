# Kernel Registries

Kernel registries define canonical indexes for ATOMIx Enterprise Asset Intelligence.

This is documentation and contract material only. It introduces no runtime registry service, database, API, executable validator, generated code, or provider adapter.

## Canonical Registries

- Asset Type Registry.
- Relationship Type Registry.
- Policy Registry.
- Capability Registry.
- Event Type Registry.
- Telemetry Type Registry.
- Lifecycle State Registry.
- Governance Registry.
- Intelligence Registry.
- Validation Rule Registry.

## Rule

Registries are contract sources of truth. Runtime may later implement or cache registry data, but runtime is never authoritative.
