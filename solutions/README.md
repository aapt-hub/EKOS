# Solutions

This folder lists optional, certified, composable EAi solution packs.

## Solution Packs

- Healthcarei
- Accountingi
- MSPi
- CRMi
- ERPi
- ITSMi
- Securityi
- Compliancei
- Financei
- Legali
- Educationi
- Governmenti
- Retaili
- Manufacturingi
- Logisticsi

## Rules

- Solution packs are deployable from `CONSOLEi` and `FACTORYi`.
- No solution pack should force the full platform to be generated for every customer.
- Solution packs remain optional until approved by governance.
