# Module Lifecycle

## Lifecycle

```text
Proposed
-> Documented
-> Structured
-> Reviewed
-> Approved
-> Generated
-> Controller Tested
-> Promoted
-> Operated
```

## Rules

- Documentation-first and structure-first are used where runtime is not yet approved.
- Generated modules must pass controller tests.
- Development -> Test -> Production promotion is strict.
- Runtime decisions require `LOSi` policy checks.

## Solution Packs

Solution packs are optional, certified, composable, and deployable from `CONSOLEi` and `FACTORYi`.
