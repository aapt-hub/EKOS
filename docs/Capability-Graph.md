# Capability Graph

Capability Graph is a first-class architectural component in EAi.

## Purpose

It connects:

- business capabilities
- architecture capabilities
- solution-pack capabilities
- governance controls
- operational evidence
- owner workflows

## Uses

- business capability governance
- lifecycle tracking
- pipeline leakage detection
- retention risk detection
- revenue/process bottleneck detection
- compliance validation
- automation recommendations
- owner notification workflows

## Relationship To Platform

- `INDEXi` provides catalog, discovery, metadata, and dependency lineage.
- `FACTORYi` uses the graph for governed generation and structure planning.
- `CONTROLi` uses the graph for workflow routing and notifications.
