# Canonical Vocabulary

This vocabulary is authoritative for ATOMIx Enterprise Asset Intelligence documentation, contracts, schemas, registries, validation notes, prompts, and certification material.

## Core Terms

- ATOMIx: Enterprise Asset Intelligence Platform.
- Mission: One Kernel. One Graph. One Enterprise.
- Enterprise Asset: any governed object the enterprise needs to understand, secure, operate, observe, or reason about.
- Kernel: contract-only foundation that owns contracts, schemas, registries, validation, and discovery.
- Enterprise Graph: central knowledge model containing assets, relationships, dependencies, capabilities, policies, events, digital twins, and telemetry.
- Runtime: replaceable implementation that consumes Kernel contracts.
- Validation: evidence-producing check that documentation, contracts, schemas, and registries are consistent.
- Certification: governance gate proving readiness against documented requirements and evidence.

## Boundary

This vocabulary introduces no runtime service, no database, no API, no endpoint agent, and no executable infrastructure.
