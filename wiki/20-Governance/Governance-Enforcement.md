# Governance Enforcement

Governance Enforcement is implemented in `los/modules/`.

It validates:

- Contract existence and identity
- Schema reference compatibility
- Attestation presence and hash match
- Runtime policy authorization
- Broker-level allow/deny behavior
- Governance provenance
- Compliance report coverage
