# LOS Governance

LOS governance is retained as legacy runtime authority material under the EAi architecture freeze.

Implemented governance chain:

```text
Contract -> Schema -> Attestation -> Policy -> Execution Broker -> Runtime Execution -> Certification -> Trust Authority -> Trust Monitoring -> Trust Enforcement
```
