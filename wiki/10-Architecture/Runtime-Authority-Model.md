# Runtime Authority Model

Runtime authority is established by a deterministic chain:

```text
Contract
-> Schema
-> Attestation
-> Policy
-> Execution Broker
-> Runtime Execution
-> Certification
-> Trust Authority
-> Trust Monitoring
-> Trust Enforcement
```

Future authority stages add recovery, trust policy, dashboard visibility, and the Runtime Authority Kernel.
